Features (big view)

  • Lexer
  • Parser
  • Interpreter
  • follows the Ecma-262 specification, is nearly compliant with it´s draft version.
  • Interfaces for code generation, decoration
  • Bytecode Interpreter soon!
  • Includes my Homepage syntax-highlighter, which has mouseover code annotations.
  • Which will be upgraded to guess the language and hit a parser from a factory.
  • Runs in Browsers, on Node, with Web Workers.
